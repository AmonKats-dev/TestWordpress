== Description ==
The Events Calendar Countdown Addon provides the ability to create Beautiful Countdown for The Events Calendar (by Modern Tribe) events with just a few clicks.

You can easily integrate event countdown anywhere(page/posts)using our plugin shortcode.
No need to setup upcoming events countdown manually.It will automatically start next upcoming event countdown after completing the current event.
It also support many advance settings.You can also choose manully muliple events for countdown.
Since version 1.2, you can view live preview of event countdown in the settings panel.
It is also translation ready you can translate all strings in your native language.

== Installation ==
<h4>Before Installation Please install <a href="http://wordpress.org/plugins/the-events-calendar/">The Events Calendar (by Modern Tribe)</a>plugin</h4>
<h6>There are 3 ways to install Countdown for The Events Calendar inside your wordpress website.</h6>
<ol>
<li><strong>Automatically install our plugin via WordPress admin panel</strong>:- Open your WordPress website admin panel and go to Plugins > Click Add New & search '<strong>Countdown for The Events Calendar</strong>' > Here you will find our plugin > Now click on install button > After this you will see a activate button > Just activate it to use Countdown for The Events Calendar inside your website</li>

<li><strong>Install via Zip file</strong>:- In this method first you need to download our plugin from wordpress plugins directory. Go to https://wordpress.org/plugins/countdown-for-the-event-calendar <strong>Countdown for The Events Calendar</strong>. After this inside your WP-admin panel, click on Plugins > Add New > Upload Button. Now choose the plugin zip file that you just downloaded from wordpress plugins directory & activate it.</li>

<li><strong>Install plugin via FTP</strong>:- In this method you also first need to download plugin zip file from wordpress plugins directory. After this open your FTP manager and Go to > wp-content/plugins folder. Here you need to upload extracted version of <strong>countdown-for-the-event-calendar</strong> (please remember don't upload zip file directly here) After this you can activate plugin from wp-admin panel plugins page.</li>
</ol>
